Apache Axis2 User's Guide

-Axis2 version 1.0

This document describes how to write Web Services and Web Service client using Axis2. It also describes how to write a custom module and engaging it within a Web Service. Samples shipped with the binary distribution of Axis2, is also discussed. It also contains a section on Advanced Topics.
