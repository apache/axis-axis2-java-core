Apache Axis2 2.0.1 Release Notes
--------------------------------

## Java Version Support

Apache Axis2 2.0.1 adds **OpenJDK 21 support** and **requires OpenJDK 17 as the minimum version** (upgraded from Java 8). The SecurityManager APIs removed in Java 21 are handled transparently, and legacy Date/Calendar APIs have been modernized to use java.time APIs. All features are fully tested and supported on both Java 17 and Java 21.

## Breaking Changes

- **Minimum Java Version**: Now requires **Java 17** (previously Java 8)
- **API Modernization**: Legacy Date/Calendar usage replaced with java.time.Instant APIs
- **Security**: SecurityManager dependent code removed for Java 21 compatibility
